Detailed Description

The Beam class consist of 3D polygon, propagation direction and it's history, Jones matrix and so on.

The diffraction calculation equations are from C. Heffels. On-line particle size and shape characterization by narrow angle light scattering. Delft University of Technology, 1995 ISBN 90-407-1181-X
